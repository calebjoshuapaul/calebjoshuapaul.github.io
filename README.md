# calebjoshuapaul.github.io

Portfolio site built with React + Vite + Three.js.

## Tooling

- Package manager/runtime: `bun`
- Lint/format: `biome`
- Build tool: `vite`
- Language mode: JavaScript + TypeScript coexistence (`allowJs`)

## Setup

```bash
bun install
```

## Scripts

```bash
bun run dev
bun run build
bun run preview
bun run lint
bun run format
```
